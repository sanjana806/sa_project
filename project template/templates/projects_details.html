Sentiment Analysis Model

Problem Statement

Businesses need to efficiently gauge customer sentiment from thousands of online reviews. This project aimed to build a machine learning model capable of automatically classifying text as either positive or negative.

My Approach

I developed a Natural Language Processing (NLP) model using a Logistic Regression algorithm. The process involved extensive text preprocessing, including tokenization and vectorization using TF-IDF, followed by training the model on a labeled dataset of movie reviews and evaluating its accuracy on unseen data.

Tools Used

Python Scikit-learn Pandas NLTK
