Data Cleaning & Transformation

The 'Date' column is an object type and needs to be converted to datetime. We'll also extract the year and month for easier time-based analysis.

In [7]:
# Convert 'Date' column to datetime objects
df['Date'] = pd.to_datetime(df['Date'])

# Extract Year and Month
df['Year'] = df['Date'].dt.year
df['Month'] = df['Date'].dt.month

Image Data Preprocessing

For computer vision tasks, images must be normalized. This involves resizing them to a consistent dimension and converting them to grayscale to reduce complexity and focus on shapes and textures.

In [9]:
# Import image processing library
import cv2

# Load an image
image = cv2.imread('path/to/your/image.jpg')

# Resize the image to 128x128 pixels
resized_image = cv2.resize(image, (128, 128))

# Convert the image to grayscale
gray_image = cv2.cvtColor(resized_image, cv2.COLOR_BGR2GRAY)
