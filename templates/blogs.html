Data Structure in Python

Data structures are ways to store and organize data in Python for easy use. List ([]) → Ordered, changeable, allows duplicates. Tuple (()) → Ordered, unchangeable, faster than lists. Set ({}) → Unordered, unique values only (no duplicates). Dictionary ({key: value}) → Stores data in key–value pairs. They make storing, searching, and managing data efficient.
